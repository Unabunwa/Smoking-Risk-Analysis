# Smoking-Risk-Analysis
A Power BI dashboard analysing patient health data, including smoking habits, BMI, and risk indicators.

# Project Overview
In many emerging healthcare systems, medical records are fragmented, making it difficult to identify high-risk patients before organ damage becomes irreversible. This project is a comprehensive Data Analytics project that examines a dataset of 2,500 patients to identify how smoking habits, BMI, and clinical markers (Cholesterol / BP) intersect to impact organ health.
